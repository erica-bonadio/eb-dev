Política de Privacidade

App: Lemon · Desenvolvedora: Erica Bonadio (EBDev) · Atualizado em maio de 2026

Esta política descreve como o sistema web Lemon trata informações de usuários e dados comerciais.

1. Quem somos

O Lemon é desenvolvido por Erica Bonadio, desenvolvedora independente (EBDev). Contato: ericacbonadio@gmail.com

2. Dados tratados

Conforme o uso do sistema, podem ser armazenados:

  • Dados de login (e-mail, nome, perfil de acesso)
  • Dados de clientes e empresas compradoras
  • Registros de produtos, lotes, estoque e vendas
  • Relatórios e histórico operacional gerados no app

3. Finalidade

Os dados são usados para operar o sistema de gestão de vendas e estoque de limões, com controle de acesso por perfil (administrador, vendedor, contadora, comprador).

4. Armazenamento

Os dados ficam em banco de dados PostgreSQL na nuvem, com hospedagem via Firebase App Hosting / infraestrutura associada. Não são armazenados apenas no navegador do usuário.

5. Compartilhamento

O desenvolvedor não vende dados pessoais. Informações de operação podem ser acessadas por usuários autorizados da equipe conforme o perfil de permissão. Provedores de infraestrutura (hospedagem e banco de dados) processam dados apenas para manter o serviço no ar.

6. Segurança e retenção

O acesso exige autenticação. Senhas são armazenadas de forma protegida (hash). Dados permanecem enquanto a conta e o serviço estiverem ativos ou conforme necessidade operacional e legal.

7. Direitos do titular (LGPD)

Para acessar, corrigir ou solicitar exclusão de dados, ou para dúvidas sobre esta política, escreva para ericacbonadio@gmail.com.
